### 주요 프로젝트

#### 1. [태영생막창] 실매장 POS 데이터 기반 수익성 개선
* **핵심**: 13개월 POS 데이터 분석을 통한 BEP(손익분기점) 산출 및 재고 최적화(ROP) 가이드 수립
* **링크**: [👉 프로젝트 상세보기](https://github.com/Kiryu-Gento/taeyoung-profit-analysis)

#### 2. [공정 분석] 화장품 품질 패턴 도출 및 제어 전략
* **핵심**: 혼합 공정 데이터 기반 품질 변수 식별 및 상관분석을 통한 최적 제어 구간 도출
* **링크**: [👉 프로젝트 상세보기](https://github.com/Kiryu-Gento/process-quality-analysis)

#### 3. [퀵커머스] 고객 행동 데이터 기반 개인화 마케팅
* **핵심**: RFM 및 연관 분석을 활용한 휴면 위험군 정의 및 구독 전환 전략 설계
* **링크**: [👉 프로젝트 상세보기](https://github.com/Kiryu-Gento/quick-commerce-personalization-analysis)
